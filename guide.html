🚀 Getting Started with MAX Bot API

1. Получите токен бота

Создайте бота в MAX и получите API токен

2. Авторизация

Добавьте токен в заголовок: Authorization: Bearer YOUR_TOKEN

3. Первый запрос

Проверьте авторизацию через эндпоинт /me

4. Начните разработку

Используйте Swagger UI для тестирования всех методов
